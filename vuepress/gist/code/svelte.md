# SvelteKit Setup

SvelteKit setup with support for TailwindCSS and Vercel/Netlift/Cloudflare.

## Create SvelteKit Project

```sh
npm init svelte@next .
```

## Add Tailwind CSS

```sh
npx svelte-add@latest tailwindcss
```

## Finalize

```sh
npm install
npm run dev -- --open
```
