# SvelteKit Setup

SvelteKit setup with support for TailwindCSS and Vercel.

## Create SvelteKit Project

```powershell
npm init svelte@next .
```

## Add Tailwind CSS

```powershell
npx svelte-add@latest tailwindcss
```

## Finalize

```powershell
npm install
npm run dev -- --open
```
